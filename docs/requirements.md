# 产品需求文档 — AI 客户咨询跟进助手

## 产品概述
一款运行在 Mac 本地的 AI 客户咨询跟进助手 Web 应用。客户通过在线表单提交咨询，系统调用 Claude API 自动分析需求、判断意向等级、生成回复邮件草稿。帮助 B 端业务人员高效管理客户咨询和跟进。

## 用户角色
- **客户**：填写咨询表单，提交需求
- **业务人员（你）**：查看 Dashboard，管理客户，导出数据

## 功能需求

### F1 — 客户咨询表单
- 输入字段：Name（必填）、Email（必填）、Company、Need、Budget、Timeline、Notes
- 表单验证：Name 和 Email 不能为空，Email 格式校验
- 提交成功后自动跳转至结果页

### F2 — AI 自动分析
- 客户提交后自动调用 Claude API 完成以下分析：
  - 客户需求总结和关键痛点识别
  - 意向等级判定：High / Medium / Low（附判断依据）
  - 生成一封专业的英文回复邮件草稿
  - 建议合理的 follow-up 日期

### F3 — Dashboard 仪表盘
- 顶部统计卡片：总客户数、High 数量、Medium 数量、Low 数量
- 客户列表表格：展示姓名、公司、意向等级（颜色标签）、Follow-up 日期、提交时间
- 按意向等级筛选（全部 / High / Medium / Low）
- 点击客户姓名进入详情页

### F4 — 客户详情
- 展示客户全部提交信息
- 展示 AI 分析结果
- 展示回复邮件草稿（支持一键复制）
- Follow-up 日期

### F5 — 数据导出
- 一键导出全部客户数据为 CSV 文件
- CSV 包含所有字段（含 AI 分析结果）

### F6 — Demo 数据生成
- 一键生成 5-8 条模拟客户数据
- 覆盖不同行业、不同预算、不同意向等级
- 每条自动触发 AI 分析

## 非功能需求
- 本地运行，无需联网部署
- 页面加载速度 < 2 秒
- AI 分析响应 < 10 秒
- UI 简洁直观，淡蓝色主色调
