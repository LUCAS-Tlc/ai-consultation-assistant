# UI 设计规范 — AI 客户咨询跟进助手

## 色彩系统

| 用途 | 颜色代码 | 色值 |
|------|----------|------|
| 主背景 | `#F0F7FF` | 极淡蓝 |
| 卡片背景 | `#FFFFFF` | 白色 |
| 主标题/按钮 | `#1976D2` | 深蓝 |
| 强调色/链接 | `#42A5F5` | 中蓝 |
| 边框/分割线 | `#E0E0E0` | 浅灰 |
| 文字主色 | `#333333` | 深灰 |
| 文字次要 | `#666666` | 中灰 |
| 成功/High | `#E53935` | 红色 |
| 警告/Medium | `#FB8C00` | 橙色 |
| 普通/Low | `#9E9E9E` | 灰色 |

## 组件规范

### 导航栏
- 高度 56px，深蓝背景 `#1976D2`
- 白色文字，左侧 Logo，右侧导航链接
- 当前页高亮

### 统计卡片
- 白色背景，圆角 8px，阴影 `0 2px 8px rgba(0,0,0,0.08)`
- 标题在顶部（灰色小字），数字在下方（大号深色字体）

### 表单
- 标签在上，输入框在下
- 输入框高度 40px，边框 `#E0E0E0`，聚焦时边框 `#42A5F5`
- 必填字段标签后有红色 `*`
- 提交按钮：深蓝背景 `#1976D2`，白色文字，圆角 4px

### 表格
- 表头浅灰背景 `#F5F5F5`
- 行间隔色（白/极淡灰交替）
- 意向等级用彩色标签显示：
  - High：红底白字
  - Medium：橙底白字
  - Low：灰底白字

### 意向等级标签
- 圆角 pill 形状，内边距 2px 12px
- High: `background: #E53935; color: white`
- Medium: `background: #FB8C00; color: white`
- Low: `background: #9E9E9E; color: white`

## 字体
- 系统默认字体栈：`-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif`
- 标题：bold, 24px
- 正文：regular, 14px
- 小字：regular, 12px

## 响应式
- 最小宽度 320px
- Mac 最佳浏览宽度 1024px-1440px
- 内容区最大宽度 1200px，居中显示
